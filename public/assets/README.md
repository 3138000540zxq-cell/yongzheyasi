## Player Walk Sheet

- Source: user-provided local character walk sheet.
- Usage: live player sprite and four-direction walking animation.
- Reference image: `./player/references/player-walk-reference.png`
- Processed sheet: `./player/player-walk-sheet.png`
- Stabilized sheet: `./player/player-walk-sheet-stabilized.png`

The reference image is a `1086 x 1448` PNG arranged as 3 columns by 4 rows. Each frame is
`362 x 362`. The source PNG does not include a real transparent alpha channel, so
`player-walk-sheet.png` was generated by removing the connected checkerboard background and
saving the result as a transparent PNG. Direction rows are used as down, left, right, and up.
`player-walk-sheet-stabilized.png` keeps the same 12-frame layout and aligns each frame to a
shared foot baseline and center point to reduce visible walking jitter.

## UI Reference Images

- Source: user-provided local design references.
- Usage: reference only for future game UI design and slicing work.
- Location: `./ui/references/`

Files:

- `ui-frame-elements.png`: frame strips, round frames, wooden corners, and panel pieces.
- `ui-icons.png`: heart, sparkle, coin, sword, book, scroll, and skull icons.
- `ui-dialog-banner.png`: long dialog/banner panel reference.
- `ui-status-panel.png`: complete RPG-style status panel reference.

These PNG files do not currently include a real transparent alpha channel. The checkerboard
background is part of the image content. If these references are used in the live game UI,
they should be sliced and processed for transparency first.

## Map HUD Images

- Source: generated from user-provided UI reference images.
- Usage: live map HUD status panel and message banner.
- Location: `./ui/hud/`

Files:

- `hud-status-panel.png`: generated from `./ui/references/ui-frame-elements.png`.
- `hud-message-banner.png`: generated from `./ui/references/ui-dialog-banner.png`.

These HUD PNG files were cropped, scaled to their in-game display size, and saved with a
transparent alpha channel so the checkerboard background is not rendered in the game.

## Character UI Images

- Source: user-provided local character attribute UI references.
- Usage: character attribute panel reference and live in-game character panel.
- Location: `./character-ui/`

Files:

- `references/character-ui-elements-reference.png`: character UI component sheet reference.
- `references/character-status-panel-reference.png`: full character status panel reference.
- `character-status-panel.png`: live character panel generated from the full status panel reference.

The reference PNG files do not include a real transparent alpha channel. The checkerboard
background is part of the image content. `character-status-panel.png` was generated by
removing the connected checkerboard background and saving the result with a transparent
alpha channel for the live Phaser character panel.
